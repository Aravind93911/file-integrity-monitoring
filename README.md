
# 🔐 File Integrity Checker

A Python tool to generate and verify file integrity using SHA-256 hashes.

## Features
- Generate and save hashes of files in a directory.
- Compare file hashes to detect modifications, additions, or deletions.

## Usage

```bash
python file_integrity_checker.py
```

Then:
1. Select option `1` to generate and save hashes.
2. Later, select option `2` to compare hashes and check integrity.

## Dependencies
- Python 3.6+

No external libraries required.
